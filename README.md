# 🎧 Mood-Based Music Finder

## 📌 Overview

Mood-Based Music Finder is a web application that helps users discover songs based on their current mood. Users can select a mood or search for songs, and the app fetches real-time music data using a public API.

---

## 🚀 Features

* 🎭 Mood Selection (Happy, Sad, Chill, Energetic)
* 🔍 Search songs by keyword
* 🎚️ Filter songs based on mood or category
* 🔼 Sort songs (A–Z / popularity)
* ❤️ Add songs to Favorites (stored in localStorage)
* 🌙 Dark Mode toggle
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML
* CSS (or Tailwind/Bootstrap)
* JavaScript (ES6)
* Fetch API
* Local Storage

---

## 🌐 API Used

* Deezer API
  Example: https://api.deezer.com/search?q=happy

---

## ⚙️ How It Works

1. User selects a mood or types in the search bar
2. The app fetches data from the API using `fetch()`
3. Songs are displayed dynamically on the UI
4. Users can:

   * Search songs
   * Sort results
   * Save favorites
